Como o bootstrap funciona


O Bootstrap tem como objetivo a construção de sites responsivos. O design responsivo garante que todos os elementos da interface funcionem seguindo o conceito mobile first, ou seja, que o design web inicialmente criado pensando em tablets e smartphones se adapte a outros dispositivos, como desktops.

O framework garante a interatividade da página graças a uma série de componentes que agilizam a navegação do usuário, como menus suspensos, controles de página, barras de progresso, etc.

O Bootstrap é uma ferramenta que permite a construção de um site a partir de modelos pré-feitos.
